1. Introduction

Bienvenue sur ONFEKOA. Nous nous engageons à protéger votre vie privée et vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons, partageons et protégeons vos informations lorsque vous utilisez notre application mobile ONFEKOA.

En utilisant notre application, vous acceptez les pratiques décrites dans cette politique de confidentialité. Si vous n'acceptez pas cette politique, veuillez ne pas utiliser notre application.

3. Données que Nous Collectons

Nous collectons différents types de données pour fournir et améliorer nos services. Voici les catégories de données que nous collectons :

3.1. Informations Personnelles

  • Nom complet : Collecté lors de l'inscription
  • Adresse e-mail : Collectée lors de l'inscription, utilisée pour l'authentification
  • Numéro de téléphone : Optionnel, collecté si vous choisissez de le fournir
  • Numéro de mobile money : Collecté si vous utilisez les paiements mobile money
  • Bio/Description : Optionnel, pour votre profil utilisateur

3.2. Données de Localisation

  • Position GPS exacte : Collectée pour trouver les activités à proximité et enregistrer la position des prestataires pour leurs activités
  • Ces données sont stockées dans notre base de données pour améliorer votre expérience

3.3. Photos et Vidéos

  • Photos de profil : Optionnel, collecté si vous choisissez d'uploader une photo
  • Photos et vidéos d'activités : Collectées par les prestataires pour illustrer leurs activités
  • Photos dans les avis : Optionnel, collecté si vous choisissez d'inclure des photos dans vos avis

3.4. Informations Financières

  • Informations de paiement : Numéro de mobile money, méthode de paiement, références de transaction
  • Historique des achats : Réservations, montants, dates de transaction
  • Autres informations financières : Montants, devises, frais de service, commissions

3.5. Contenu Généré par l'Utilisateur

  • Activités créées : Titre, description, prix, catégorie, localisation, médias
  • Réservations : Créées par les utilisateurs lors de la réservation d'activités
  • Avis et notes : Commentaires, notes, photos laissés par les utilisateurs
  • Messages : Conversations et messages entre utilisateurs et prestataires

3.6. Données d'Activité

  • Interactions avec l'application : Vues d'activités, likes, favoris, historique de connexion
  • Autres actions : Réservations, annulations, validations

3.7. Identifiants d'Appareil

  • Token de notification push : Pour envoyer des notifications
  • User Agent : Informations sur l'appareil et le navigateur
  • Informations sur l'appareil : Nom, type, système d'exploitation
  • Adresse IP : Pour la sécurité et le suivi des sessions

3.8. Données de Performance

  • Journaux de plantage : Collectés automatiquement pour améliorer la stabilité
  • Données de performance : Statistiques d'utilisation, métriques de performance

4. Comment Nous Utilisons Vos Données

Nous utilisons vos données pour les finalités suivantes :

4.1. Fonctionnement de l'Application

  • Authentifier et gérer votre compte utilisateur
  • Permettre la réservation d'activités
  • Faciliter la communication entre utilisateurs et prestataires
  • Traiter les paiements et gérer les transactions
  • Afficher les activités et les profils utilisateurs

4.2. Gestion des Comptes

  • Créer et gérer votre compte utilisateur
  • Personnaliser votre profil
  • Gérer vos réservations et votre historique

4.3. Communications

  • Envoyer des notifications push importantes
  • Envoyer des emails de confirmation et de notification
  • Vous informer des mises à jour de sécurité

4.4. Sécurité et Prévention des Fraudes

  • Vérifier l'identité des utilisateurs
  • Détecter et prévenir les fraudes
  • Assurer la sécurité des transactions
  • Respecter les obligations légales

4.5. Personnalisation

  • Recommander des activités basées sur vos préférences
  • Personnaliser votre expérience utilisateur
  • Améliorer nos services

4.6. Analyse et Amélioration

  • Analyser l'utilisation de l'application
  • Identifier et corriger les bugs
  • Améliorer les performances et la stabilité

5. Partage de Données avec des Tiers

Nous partageons vos données avec les services tiers suivants pour fournir nos services :

5.1. Supabase

  • Type de service : Backend as a Service (BaaS)
  • Données partagées : Toutes les données utilisateur collectées
  • Usage : Stockage des données, authentification, stockage de fichiers
  • Politique de confidentialité : https://supabase.com/privacy

5.2. FeexPay

  • Type de service : Passerelle de paiement
  • Données partagées : Informations financières, montants, références de transaction
  • Usage : Traitement des paiements mobile money
  • Politique de confidentialité : https://feexpay.me

5.3. Google Maps

  • Type de service : Service de cartographie
  • Données partagées : Coordonnées GPS, adresses
  • Usage : Affichage des cartes et géolocalisation
  • Politique de confidentialité : https://policies.google.com/privacy

5.4. Expo

  • Type de service : Service de notifications
  • Données partagées : Token de notification push
  • Usage : Envoi de notifications push aux utilisateurs
  • Politique de confidentialité : https://expo.dev/privacy

6. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données personnelles :

  • Chiffrement : Toutes les communications sont chiffrées via HTTPS/TLS
  • Authentification sécurisée : Système d'authentification sécurisé via Supabase Auth
  • Stockage sécurisé : Données stockées de manière sécurisée sur Supabase
  • Paiements sécurisés : Traitement sécurisé via FeexPay conforme aux standards de sécurité
  • Accès restreint : Accès aux données limité aux personnes autorisées uniquement

7. Conservation des Données

Nous conservons vos données personnelles aussi longtemps que nécessaire pour :

  • Fournir nos services
  • Respecter nos obligations légales
  • Résoudre les litiges
  • Faire respecter nos accords

Vous pouvez demander la suppression de vos données à tout moment en nous contactant. Nous supprimerons vos données dans les délais légaux applicables, sauf si nous sommes tenus de les conserver pour des raisons légales.

8. Vos Droits

Conformément aux lois sur la protection des données, vous avez les droits suivants :

  • Droit d'accès : Vous pouvez demander une copie de vos données personnelles
  • Droit de rectification : Vous pouvez corriger vos données inexactes
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données
  • Droit à la portabilité : Vous pouvez demander le transfert de vos données
  • Droit d'opposition : Vous pouvez vous opposer au traitement de vos données
  • Droit de limitation : Vous pouvez demander la limitation du traitement

Pour exercer ces droits, veuillez nous contacter à l'adresse indiquée dans la section "Contact".

9. Données des Enfants

Notre application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans. Si nous apprenons qu'un enfant de moins de 13 ans nous a fourni des données personnelles, nous supprimerons ces informations dans les plus brefs délais.

10. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre. Nous vous informerons de tout changement en publiant la nouvelle politique de confidentialité sur cette page et en mettant à jour la date de "Dernière mise à jour".

Nous vous encourageons à consulter régulièrement cette politique de confidentialité pour rester informé de la manière dont nous protégeons vos données.
